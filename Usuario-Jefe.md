# Usuario Jefe
Hereda Usuario
## 1. Login
Login único con correo institucional y contraseña asignada.

## 2. Incidencias
### 2.1. Tabla de incidencias
Aparecera una tabla con una lista de todas las incidencias del usuario
### 2.2. Crear incidencia
Cada incidencia le pedira lo siguiente: 
- Poder seleccionar el tipo de equipo
- id_usuario_reporta (Se agrega automaticamente)
- descripcion
## 3 Solicitudes
### 3.1. Tabla de Solicitudes
Aparecera una tabla con una lista de todas las Solicitudes del usuario
### 3.2. Crear Solicitudes
Cada Solicitud le pedira lo siguiente: 
- Poder seleccionar el tipo de solicitud
- id_usuario_reporta (Se agrega automaticamente)
- descripcion
